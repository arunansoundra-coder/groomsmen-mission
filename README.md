# 🎩 Operation Always and Forever  
## Groomsmen Mission Microsite (MI6 Poker Experience)

A cinematic, James Bond–inspired interactive microsite that transforms a wedding groomsmen invitation into a **mission-based authentication experience** leading into a high-stakes poker game.

---

# 🧠 Project Overview

This is a **single-page web application** hosted on GitHub Pages.

Users are invited via personalized links and guided through:

1. Identity Authentication (Bond-style questions)
2. Security Clearance Verification
3. Mission Briefing (role-based invitation)
4. Poker Table Experience (final stage)
